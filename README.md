# mind-palette

생각 팔레트 : 자신만의 색을 찾기 위한 프로젝트
===================================

개요
---
진로계획이나 대학진학에 도움이 되는 웹 (자격증 소개, 추천, 입시관련 내용, 또는 직업)


카테고리
-----
자격증 / 전공 / 입시 / 직업


요구사항
------

- Python 3.6+

설치 방법
-------

``pyvenv`` 나 ``virtualenvwrapper`` 사용을 권장합니다.


```
   $ ./install.sh
```


실행 방법
-------

웹 서버 실행 방법은 다음과 같습니다
```
   $ python run.py -p 2028
```
